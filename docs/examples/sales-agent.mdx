# Sales Agent

An Agno-oriented example that combines storage, knowledge, and channels.

### Definition: `agents/sales.yaml`
```yaml
name: Acme Sales
personality: Enthusiastic closer.
storage:
  type: sqlite
knowledge:
  type: lancedb
  sources: ["https://acmecorp.com/catalog"]
channels:
  - type: slack
    token_env: SLACK_TOKEN
```

### Skill: `skills/pricing/tools.py`
```python
def quote(price: float, qty: int) -> str:
    """Calculates discounted price."""
    return f"Total: ${price * qty * 0.9 if qty > 10 else price * qty}"
```
