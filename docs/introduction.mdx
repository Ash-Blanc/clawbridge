# clawbridge

`clawbridge` helps you build OpenClaw-style agents inside frameworks you already want to use.

If you are new to the project, think of it like this:

1. Scaffold an OpenClaw workspace
2. Define an agent in `agent.yaml` or `agents.yaml`
3. Add skills and workspace files
4. Build for Agno or Agentica
5. Add framework-specific extras only when needed

## Start here

Follow this path in order:

1. Quickstart
2. Your First Agent
3. Add Skills
4. Add Memory

The main objects you will use first are:

- `ClawAgent`
- `ClawSkill`
- `ClawMemory`
- `build_agno_agent()`
- `build_agentica_agent()`

## Fastest path

If you want the fastest path:

- scaffold one workspace
- run one small agent
- load one skill
- store one memory value

Do not start with architecture or framework-specific deployment details unless you already need them.

## What comes later

There are more advanced parts of the project, including optional Agno deployment helpers, but they are intentionally not the starting point of the docs.
