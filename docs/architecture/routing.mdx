# Routing

Any `.yaml` in `agents/` becomes a deployed native Agno `Agent`.

### Example: `agents/bot.yaml`
```yaml
name: Bot
personality: You are witty.
model:
  provider: openai
  model_id: gpt-4o
storage:
  enabled: true
  type: sqlite
```

Run `clawbridge dev` to deploy.
