# Channels

Channels are currently an Agno-oriented extension with a shared policy layer for DM, group, mention, and heartbeat behavior.

Configure them in the agent definition:

```yaml
channels:
  - type: slack
    token_env: SLACK_BOT_TOKEN
  - type: whatsapp
    token_env: WHATSAPP_TOKEN
```

Tokens are read from environment variables.

Today, treat transport execution as partial Agno-specific functionality.
