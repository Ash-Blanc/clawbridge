# Quickstart

### 1. Install
```bash
uv pip install clawbridge[agno]
```

### 2. Initialize
```bash
clawbridge init my-agent
cd my-agent
```

### 3. Set API Key
```bash
export OPENAI_API_KEY="sk-..."
```

### 4. Start Dev Server
```bash
clawbridge dev
```

Your AgentOS API is live at `http://127.0.0.1:8000`.
