# Quickstart

## What you will build

By the end of this page, you will have:

- one OpenClaw workspace
- one running agent
- one model connection
- one place to add memory and skills next

### 1. Install

```bash
uv sync --extra all
```

### 2. Set your API key

```bash
export ANTHROPIC_API_KEY="sk-..."
```

### 3. Create a workspace

```bash
clawbridge scaffold ./my-workspace
cd ./my-workspace
```

This gives you:

- the standard workspace files like `AGENTS.md`, `MEMORY.md`, and `BOOTSTRAP.md`
- a starter skill in `skills/hello_world/`
- an `agent.yaml` that `clawbridge` can load directly

### 4. Run the starter agent

```python
from clawbridge import build_agno_agent, load_agent_config

agent = load_agent_config("./agent.yaml")
native_agent = build_agno_agent(agent)
response = native_agent.run("What can you do?")
print(getattr(response, "content", response))
```

## What you just built

- a `ClawAgent`
- a workspace-backed prompt
- a native Agno agent
