# Skills

Skills are OpenClaw-style packages. In a workspace, they are usually discovered from `skills/`, but you can also attach them explicitly from Python.

### 1. Code-First (Recommended)

Create a directory with `tools.py`. No extra config is required.

```python
def add(a: int, b: int) -> int:
    """Adds two numbers."""
    return a + b
```

### 2. OpenClaw Standard

Add `SKILL.md` for human-readable instructions and explicit YAML schemas.

If a skill defines only metadata, provide the implementation separately through `ClawAgent.tools`.
