# Structure

`clawbridge` uses file-system routing.

| Path | Purpose |
| :--- | :--- |
| `agents/*.yaml` | Auto-deployed agents. |
| `skills/*/tools.py` | Auto-mounted Python tools. |
| `knowledge/` | Documents for RAG. |
| `claw.config.yaml` | Global server config. |
