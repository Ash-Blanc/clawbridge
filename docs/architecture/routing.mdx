# Routing

Any `.yaml` in `agents/` becomes a deployed native Agno `Agent` in app mode.

### Example: `agents/bot.yaml`
```yaml
name: Bot
personality: You are witty.
model:
  provider: openai
  model_id: gpt-4o
storage:
  enabled: true
  type: sqlite
```

Run `clawbridge dev` to deploy through Agno `AgentOS`.

This routing model is not the main portable runtime API. It is the filesystem-driven Agno workflow.
