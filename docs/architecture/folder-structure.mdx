# Folder Structure

This layout belongs to the optional Agno deployment helper.

| Path | Purpose |
| :--- | :--- |
| `agents/*.yaml` | Agent definitions compiled into Agno agents. |
| `skills/*/tools.py` | Auto-mounted Python tools. |
| `skills/*/SKILL.md` | OpenClaw-style skill metadata and instructions. |
| `knowledge/` | Documents intended for Agno knowledge ingestion. |
| `claw.config.yaml` | Agno deployment helper server configuration. |

If you are using the default workspace-first flow, you do not need this layout.
