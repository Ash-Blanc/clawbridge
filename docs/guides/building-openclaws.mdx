# Building Portable Agents

OpenClaw-inspired agents in `clawbridge` are defined by:

1. **Identity** via `ClawAgent`
2. **Prompt assembly** via `build_system_prompt()`
3. **Capabilities** via `ClawSkill` and `ToolDefinition`
4. **Memory** via `ClawMemory`
5. **Runtime choice** via `ClawBridge`

Start with the portable core first. Add backend-specific features only when you need them.
