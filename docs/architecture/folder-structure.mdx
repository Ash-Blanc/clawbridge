# Folder Structure

This layout belongs to the optional **Agno app mode**.

| Path | Purpose |
| :--- | :--- |
| `agents/*.yaml` | Agent definitions compiled into Agno agents. |
| `skills/*/tools.py` | Auto-mounted Python tools. |
| `skills/*/SKILL.md` | OpenClaw-style skill metadata and instructions. |
| `knowledge/` | Documents intended for Agno knowledge ingestion. |
| `claw.config.yaml` | App-mode server configuration. |

If you are using the portable core directly, you do not need this layout.
