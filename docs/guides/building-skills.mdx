# Skills

Skills are auto-mounted from the `skills/` directory.

### 1. Code-First (Recommended)
Just create `skills/my_skill/tools.py`. No config needed.

```python
def add(a: int, b: int) -> int:
    """Adds two numbers."""
    return a + b
```

### 2. OpenClaw Standard
Add `SKILL.md` for human-readable instructions and explicit YAML schemas.
