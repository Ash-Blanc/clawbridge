# Quickstart

By the end of this page you'll have a workspace-backed agent with persistent memory running on Agno.

## 1. Install

<CodeGroup>
```bash pip
pip install "clawbridge[agno]"
```

```bash uv
uv add "clawbridge[agno]"
```
</CodeGroup>

## 2. Set your API key

```bash
export ANTHROPIC_API_KEY="sk-..."
```

Agno reads `ANTHROPIC_API_KEY` from the environment automatically — no extra config needed.

## 3. Scaffold a workspace

```bash
clawbridge scaffold ./my-workspace
cd my-workspace
```

This creates:

```
my-workspace/
├── AGENTS.md          # Agent identity and behavioral rules
├── MEMORY.md          # Memory conventions for the workspace
├── BOOTSTRAP.md       # Startup instructions and context
├── agent.yaml         # Starter agent definition
└── skills/
    └── hello_world/   # A working example skill
        ├── SKILL.md
        └── tools.py
```

All three markdown files are loaded into the agent's system prompt automatically when you run with `workspace_path: .`.

## 4. Run the starter agent

```python
from clawbridge import build_agno_agent, load_agent_config

agent = load_agent_config("./agent.yaml")
native_agent = build_agno_agent(agent)
native_agent.print_response("What can you do?")
```

`load_agent_config` reads the YAML, resolves the workspace files, and returns a `ClawAgent`. `build_agno_agent` compiles it into a native Agno `Agent`.

## 5. Enable persistent memory

Open `agent.yaml` and add the storage and memory fields:

```yaml
name: Assistant
workspace_path: .
personality: Pragmatic, concise, and reliable.
model:
  provider: anthropic
  model: claude-sonnet-4-20250514
storage:
  enabled: true
  type: sqlite
  db_url: agent.db
agent_memory_mode: automatic
```

Re-run the agent:

```python
from clawbridge import build_agno_agent, load_agent_config

agent = load_agent_config("./agent.yaml")
native_agent = build_agno_agent(agent)
native_agent.print_response("My name is Alex. Remember that.")

# In a new session:
native_agent.print_response("What's my name?")
# → "Your name is Alex."
```

The agent now extracts facts from conversations and recalls them across sessions using `agent.db`.

## What you just built

- A `ClawAgent` backed by a workspace directory
- Workspace files (`AGENTS.md`, `MEMORY.md`, `BOOTSTRAP.md`) compiled into the system prompt
- A hello_world skill loaded automatically from `skills/`
- Persistent SQLite memory — the agent remembers across sessions
