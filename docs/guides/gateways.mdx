# Channels

Channels are currently an Agno-oriented extension, not a portable guarantee across all backends.

Configure them in the agent definition:

```yaml
channels:
  - type: slack
    token_env: SLACK_BOT_TOKEN
  - type: whatsapp
    token_env: WHATSAPP_TOKEN
```

Tokens are read from environment variables.

Today, treat this as partial backend-specific functionality.
