# Agno Deployment Helper

This is the optional, filesystem-driven Agno deployment helper built around `clawbridge.app.ClawApp`.

It scans a project layout like this:

```text
agents/*.yaml
skills/*/SKILL.md
skills/*/tools.py
knowledge/
claw.config.yaml
```

## When to use it

Use it when:

- your deployment target is Agno
- you want project scaffolding and filesystem discovery
- you prefer convention over explicit Python wiring

## When not to use it

Do not treat this helper as the main `clawbridge` API.

If you want the default `clawbridge` flow, start with:

- `ClawAgent`
- `ClawSkill`
- `clawbridge scaffold`
- `build_agno_agent()` or `build_agentica_agent()`

Only reach for it if an Agno filesystem deployment flow is useful for your project.
