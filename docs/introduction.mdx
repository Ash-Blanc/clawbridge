# 🦞 clawbridge

**clawbridge** is a meta-framework for AI agents. Inspired by Next.js, it uses **Convention over Configuration** to eliminate orchestration boilerplate.

- **Define Once:** Declarative agent "souls" in YAML.
- **Deploy Natively:** 1:1 Agno integration with zero abstraction tax.
- **Portable Skills:** Python-first auto-mounted tools.
- **Memory:** Native persistent storage (SQLite/Postgres).

*Build once. Deploy natively. Remember everything.*
