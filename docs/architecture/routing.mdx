# Routing

Any `.yaml` in `agents/` becomes a deployed native Agno `Agent` in the optional deployment helper.

### Example: `agents/bot.yaml`

```yaml
name: Bot
personality: You are witty.
model:
  provider: openai
  model_id: gpt-4o
storage:
  enabled: true
  type: sqlite
```

Run `clawbridge dev` to deploy through Agno `AgentOS`.

This routing model is not the main `clawbridge` API. It belongs to the optional filesystem-driven Agno deployment helper.
