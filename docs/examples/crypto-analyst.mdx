# Crypto Analyst

An example that combines knowledge and WhatsApp channel config.

### Definition: `agents/alpha.yaml`
```yaml
name: AlphaTracker
model:
  provider: anthropic
  model: claude-3-5-sonnet
knowledge:
  type: lancedb
channels:
  - type: whatsapp
    token_env: WHATSAPP_TOKEN
```
