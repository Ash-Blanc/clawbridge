# clawbridge

`clawbridge` is a Python framework for building AI agents with a small, readable set of primitives.

If you are new to the project, think of it like this:

1. Create an agent
2. Run it
3. Add skills
4. Add memory
5. Explore advanced features later

## Start here

Follow this path in order:

1. Quickstart
2. Your First Agent
3. Add Skills
4. Add Memory

The main objects you will use first are:

- `ClawAgent`
- `ClawBridge`
- `ClawSkill`
- `ClawMemory`

## Fastest path

If you want the fastest path:

- read the quickstart
- build one small agent
- load one skill
- store one memory

Do not start with architecture or backend details unless you already need them.

## What comes later

There are more advanced parts of the project, including backend-specific helpers and filesystem-driven app mode, but they are intentionally not the starting point of the docs.
