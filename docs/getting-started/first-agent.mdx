# Your First Agent

Start with the smallest working version:

## Goal

Build one agent that can answer a prompt successfully.

```python
from clawbridge import ClawAgent, ModelConfig, build_agno_agent

agent = ClawAgent(
    name="Molty",
    description="A helpful assistant",
    personality="Helpful and concise.",
    model=ModelConfig(
        provider="anthropic",
        model_id="claude-sonnet-4-20250514",
        api_key_env="ANTHROPIC_API_KEY",
    ),
)

native_agent = build_agno_agent(agent)
response = native_agent.run("Say hello.")
print(getattr(response, "content", response))
```

## What each part does

- `ClawAgent` defines the assistant
- `ModelConfig` tells it which model to use
- `build_agno_agent()` returns a native Agno agent
