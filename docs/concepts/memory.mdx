# Memory

There are two memory layers in `clawbridge`:

## `ClawMemory`

`ClawMemory` is the lightweight memory helper used by `clawbridge` prompt assembly and runtime wrappers.

It provides:

- long-term key/value memory
- short-term conversation history
- prompt-time memory summaries

## Framework-native storage

Some frameworks can also map `ClawAgent.storage` into native storage primitives.

Today:

- Agno has native storage support
- Agentica does not

Use `ClawMemory` for prompt-time and lightweight state, and use native storage only when your target framework supports it.
